This is a Typst project for a cheat sheet regarding musical notation. 

# IMPORTANT
When working on `.typ` files, use the $typst skill!

# Project layout
- `main.typ` as entry point for the document

Create additional files that are imported or included into the `main.typ` whenever needed.
Do not create too long files. Create a sensible file structure.

# Visual check
After changing and compiling the file. Get screenshots of the pages and analyze the visual quality. Fix any overlapping text and other layout issues.
